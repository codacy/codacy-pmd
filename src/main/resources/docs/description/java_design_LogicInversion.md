Use opposite operator instead of negating the whole expression with a logic complement operator.
It makes code easier to read and understand.

Ex:

```
public boolean bar(int a, int b) {

  if (!(a == b)) { // use !=
    return false;
  }

  if (!(a < b)) { // use >=
    return false;
  }

  return true;
}
```
