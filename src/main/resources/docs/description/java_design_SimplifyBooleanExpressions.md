Avoid unnecessary comparisons in `boolean` expressions, they serve no purpose and impacts readability.

Ex:

```
public class Bar {
  // can be simplified to bar = isFoo();
  private boolean bar = (isFoo() == true);

  public isFoo() {
    return false;
  }
}
```
